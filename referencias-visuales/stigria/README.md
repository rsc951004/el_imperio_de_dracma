# Boceto geográfico histórico de Stigria

## Referencia vigente

`boceto-stigria.png` es el boceto geográfico de trabajo vigente. El usuario lo actualiza directamente; no se mantiene una cadena de versiones numeradas.

La imagen representa regiones físicas e históricas, no reinos ni fronteras políticas. Incluye el Gherin, el Grimm, Stomur, las regiones gheran y los principales sistemas acuáticos.

Ante cualquier diferencia entre la imagen y la documentación, prevalece el canon escrito en `docs/`.
