# Referencias visuales

Esta carpeta conserva imágenes tentativas para orientar la exploración visual de «El Imperio de Dracma».

## Condición editorial

- Ninguna imagen de esta carpeta es canónica.
- Las imágenes no sustituyen ni modifican el lore escrito en `docs/` y `cuentos/`.
- Si una imagen contradice el canon, prevalece siempre el contenido de `docs/` y `cuentos/`.
- Cualquier detalle visual debe canonizarse por escrito antes de considerarse real dentro del universo.

## Contenido

- `mapa-tentativo-estigria.png`: interpretación visual preliminar de Estigria basada en el tablero de El Gerin y las fichas geográficas canónicas disponibles al momento de su creación.
