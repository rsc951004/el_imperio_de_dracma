# Kantra

## Tipo

Palabra o acción utilizada por algunos Portadores para activar o canalizar con
mayor precisión su Blesk.

## Ejemplo documentado

Durante la Guerra de Unland, Roshan Therin empleó la Kantra **«Ta-besh»** al
manifestar su fulgor solar.
