# Gaizka Darerca

## Identidad
Rey consorte de **Lumaria**. Originario de **Ylvaria** (Casa Darerca).

- **Reino actual:** Lumaria (rey consorte).
- **Reino de origen:** Ylvaria (bosques, pueblo Ylum).
- **Cultura:** Gaélico/Vasco.

## Relaciones
- **Dreide Gorio:** hermana. Reina gobernante de Ylvaria.
- **Ezria Maimon:** esposa. Reina gobernante de Lumaria.
