# Titia Corvus

## Identidad
Reina consorte de **Ylvaria**. Originaria de **Aurelia** (Casa Corvus).

- **Reino actual:** Ylvaria (reina consorte).
- **Reino de origen:** Aurelia (centro-sur fértil, herencia Erebia).
- **Cultura:** Latín.

## Relaciones
- **Valerius Umbra:** hermano. Rey gobernante de Aurelia.
- **Dreide Gorio:** esposa. Reina gobernante de Ylvaria.
