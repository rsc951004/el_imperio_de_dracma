# Índice de personajes

Cada personaje canónico posee una ficha individual en la carpeta `personajes/`.

## Formato de las fichas

```markdown
# Nombre

## Identidad
[Quién es, qué hace, a qué casa o grupo pertenece.]

## Personalidad
[Rasgos esenciales. No historia: carácter.]

## Relaciones
[Personajes vinculados. Nombre + naturaleza del vínculo.]

## Referencias
[Cuentos o protoversiones donde aparece.]
```

> Las secciones adicionales se añaden según el personaje lo requiera. La ficha es una herramienta de consulta para escritores: seca, rápida, sin prosa.
