# El Imperio de Dracma

Colección de cuentos **creada íntegramente con inteligencia artificial**.
Kilo es el orquestador principal y los agentes IA son los autores, editores y mantenedores del lore.

---

## Para agentes IA

Este proyecto está diseñado para ser operado por agentes. Cada carpeta contiene su
propio `AGENTS.md` con reglas específicas. El archivo raíz [`AGENTS.md`](AGENTS.md)
define las reglas universales. **Leer antes de trabajar.**

| Carpeta | Guía para agentes | ¿Canon? |
|---------|-------------------|----------|
| `cuentos/` | [`cuentos/AGENTS.md`](cuentos/AGENTS.md) | ✅ |
| `docs/` | [`docs/AGENTS.md`](docs/AGENTS.md) | ✅ |
| `protoversiones/` | [`protoversiones/AGENTS.md`](protoversiones/AGENTS.md) ⛔ | ❌ |
| `herramientas/` | [`herramientas/AGENTS.md`](herramientas/AGENTS.md) | N/A |

---

## ⚠️ Canon vs. Prototipo

| Zona | Carpeta | ¿Es canon? | ¿Se publica? |
|------|---------|-------------|---------------|
| **Canon** | `cuentos/` + `docs/` | ✅ Sí | ✅ Sí |
| **Cantera** | `protoversiones/` | ❌ No | ❌ Jamás |

`protoversiones/` contiene borradores antiguos. Se conservan solo para rescatar ideas
puntuales durante la fase de depuración. **Nada en esa carpeta es parte del lore real.**
